# Website For Board

如果你的设备具有联网功能，那么为什么不在里面尝试建个网站，用动态网站去控制你的设备呢。参考文档主要是用了Busybox httpd，其实用其他的网站服务器也是类似，只是Busybox httpd最简单。TI AM335x就赋予了这个功能：[Matrix Users Guide](http://processors.wiki.ti.com/index.php/Matrix_Users_Guide)。
